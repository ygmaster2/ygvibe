programs.csv 사용법
====================

이 파일은 대시보드(dashboard/index.html)가 읽어서 카드로 보여주는 데이터입니다.
엑셀, 한셀, 구글시트, 메모장 등 어떤 프로그램으로 열어도 되지만,
저장할 때는 반드시 "CSV(쉼표로 분리)" 형식으로 저장해주세요 (xlsx로 저장하면 안 됩니다).

컬럼 설명
---------
- 프로그램명   : 카드 제목
- 뱃지         : (선택) 카드 상단에 표시할 짧은 태그. 비워두면 표시 안 됨
- 제작교사     : (선택) 예) 수학과 · 김OO 교사
- 설명         : 카드에 표시할 2~3문장 설명
- 바로가기유형 : "폴더" 또는 "URL" 중 하나를 정확히 입력
    · 폴더 : 단일 html 파일로 된 프로그램을 저장소에 올려서 바로 열 때.
             바로가기값에 programs 폴더 안 파일명만 입력 (매뉴얼파일명과 같은 방식)
             예) 내프로그램.html  →  실제 파일 위치는 programs/내프로그램.html
    · URL  : 외부에 이미 호스팅된 프로그램/사이트로 연결할 때.
             바로가기값에 전체 주소(https://...) 입력
- 바로가기값   : 위 유형에 맞는 파일명 또는 URL
- 매뉴얼파일명 : manuals 폴더 안에 있는 매뉴얼 html 파일명만 입력 (예: 내프로그램.html)

새 프로그램이 생기면
---------------------
1. (폴더 방식인 경우) 프로그램 html 파일을 programs/ 폴더에 정확한 이름으로 추가
2. 매뉴얼 html 파일을 manuals/ 폴더에 정확한 이름으로 추가
3. programs.csv에 새 행 추가 (파일명들을 그대로 입력) 후 CSV 형식으로 저장
4. 이 CSV 파일 + 위 파일들을 GitHub 저장소에 업로드
5. 잠시 후 GitHub Pages 사이트에 카드가 자동으로 반영됨

주의사항
--------
- 반드시 CSV(쉼표로 분리) 형식으로 저장하세요. xlsx로 저장하면 프로그램(특히 한셀)에 따라
  대시보드가 읽지 못할 수 있습니다.
- 설명 등 텍스트 안에 쉼표(,)가 들어가도 대부분의 프로그램이 자동으로 따옴표 처리를 해주지만,
  이상하게 깨진다면 그 필드를 큰따옴표(")로 감싸주세요.
