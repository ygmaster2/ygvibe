시스템 개요

사정안 자동화 시스템 베타 버전 사용자 안내서

🎯 이 시스템은 무엇인가요?

사정안 자동화 시스템은 학기말 사정안 작성 및 교과우수상 자료 수집 업무를 디지털화하여 담임교사·교과교사의 입력 작업과 관리자의 취합·출력 업무를 효율화하는 웹 기반 시스템입니다. Google Apps Script(GAS) 환경에서 구동되며, 인터넷 브라우저만 있으면 별도 설치 없이 사용 가능합니다.

ℹ️ Chrome 브라우저 사용을 권장합니다. 모바일 환경에서는 일부 기능이 제한될 수 있습니다.

👥 사용자 역할

🔐

관리자

시스템 설정, 제출 현황 모니터링, 데이터 내보내기 등 전체 업무를 총괄합니다.

👨‍🏫

담임 교사

담당 학급의 재적 현황, 표창 대상자 등 사정안 양식을 작성하고 제출합니다.

📖

교과 교사

나이스에서 받은 교과우수상 대상자 xlsx 파일을 업로드합니다.

관리자 매뉴얼

시스템 전반을 설정하고 제출 현황을 관리합니다

🔑 관리자 로그인

  1. 메인 화면에서 🔐 관리자 로그인 버튼을 클릭합니다.
  2. 비밀번호를 입력하고 로그인 버튼을 클릭합니다.
  3. 로그인 성공 시 관리자 대시보드 화면으로 이동합니다.
⚠️ 비밀번호는 Google Sheets의 master 시트 A2 셀에 저장되어 있습니다. 분실 시 스프레드시트에서 직접 확인하세요.

⚙️ 시스템 설정

관리자 대시보드 우상단 ⚙️ 시스템 설정 버튼을 클릭하면 설정 창이 열립니다.

학년도 현재 학년도를 입력합니다. (예: 2026)
담임교사 양식 상단에 표시됩니다.
현재 학기 1학기 선택 시 — 담임교사의 2학기 탭이 잠깁니다.
2학기 선택 시 — 담임교사의 2학기 탭이 활성화됩니다.
담임교사 로그인 토글 ON: 담임교사 로그인 허용  |  OFF: 로그인 잠금
과목교사 로그인 성적 산출 완료 후 ON으로 변경합니다.
OFF 상태에서는 교과교사 로그인 버튼이 비활성화됩니다.
✅ 저장 버튼을 누르면 즉시 모든 사용자에게 반영됩니다. 재배포 없이 바로 적용됩니다.
💡 배포 전에 학년도·학기를 먼저 설정해두세요. 담임교사와 교과교사가 접속하면 설정값이 자동 적용됩니다.

📊 대시보드 활용

① 학기 탭 전환

상단 탭에서 📚 1학기 사정안 / 📚 2학기(학년말) 사정안을 선택하여 학기별 현황을 확인합니다.

② 통계 영역

📋 담임자료 통계

전체 반 수 · 제출 완료 · 미제출 반 수를 한눈에 확인

📚 교과자료 통계

전체 과목 수 · 업로드 완료 과목 수 표시

③ 담임 카드

학년별로 1~8반 카드가 나열됩니다. 카드 색상으로 상태를 확인할 수 있습니다.

✓ 제출됨  제출 완료 ※ 임시저장  작성 중 ✗ 미제출  미작성
ℹ️ 카드를 클릭하면 해당 반의 사정안 내용을 전체 화면으로 조회할 수 있습니다.

④ 교과 카드

학년별로 과목편제에 등록된 과목들이 카드로 표시됩니다. 카드를 클릭하면 업로드된 학생 명단을 확인할 수 있습니다.

✅ 과목명  업로드 완료 + 인원수 표시 ❌ 과목명  미업로드

⑤ 담임 화면 조회

  1. 상단 🔍 담임 데이터 조회 영역에서 학년·반을 선택합니다.
  2. 📋 조회 버튼을 클릭하면 해당 반의 사정안을 전체 화면으로 볼 수 있습니다.
  3. 모달 상단 🖨️ 인쇄 버튼으로 출력 가능합니다.

⑥ 학년별 전체 출력

  1. 🖨️ 학년별 전체 출력 영역에서 원하는 학년 버튼을 클릭합니다.
  2. 해당 학년 전체 반의 사정안이 A4 가로 형식으로 연속 표시됩니다.
  3. 🖨️ 인쇄 버튼으로 한 번에 출력합니다.

📤 데이터 내보내기 및 한글 문서 생성

한글(HWP) 문서 생성은 로컬 PC의 converter.exe를 통해 이루어집니다. 아래 순서대로 진행하세요.

① 데이터 전송 (양식 시트 준비)

  1. 대시보드 하단 📤 데이터 전송 버튼을 클릭합니다.
  2. 현재 활성 탭(1학기 또는 2학기) 기준으로 담임 자료가 양식_1학기 또는 양식_2학기 시트에 자동으로 채워집니다.
  3. 전송 완료 메시지가 뜨면 구글 시트에서 내용을 확인할 수 있습니다.
💡 양식 시트는 추후 엑셀 출력 등 다양한 용도로 활용될 수 있습니다. 현재는 데이터를 정해진 양식 형태로 정리하는 기능을 합니다.

② 사정안 한글문서 생성 (메일머지)

  1. 담임 교사와 교과 교사의 자료 업로드가 완료된 것을 확인합니다.
  2. 대시보드 하단 📋 1학기 CSV 내보내기 또는 📋 2학기 CSV 내보내기 버튼을 클릭하여 CSV 파일을 다운로드합니다.
  3. 관리자 PC에서 converter.exe를 실행합니다.
  4. converter.exe에서 다운로드된 CSV 파일을 불러와 메일머지 실행 버튼을 클릭합니다.
  5. 교과우수상 HWP 문서가 생성됩니다.
💡 converter.exe는 관리자 PC에서만 실행하면 됩니다. 교사들은 웹 브라우저만으로 데이터를 입력·제출할 수 있습니다.
👨‍🏫

담임 교사 매뉴얼

학급 사정안을 작성하고 제출합니다

🔑 로그인

  1. 메인 화면에서 👨‍🏫 담임 교사 로그인 버튼을 클릭합니다.
    ⚠️버튼이 회색(비활성화)으로 표시되면 관리자가 아직 로그인을 열지 않은 상태입니다. 관리자에게 문의하세요.
  2. 학년을 선택합니다. (1학년 / 2학년 / 3학년)
  3. 반을 선택합니다. (1반 ~ 8반)
  4. 교사명을 입력합니다. (class_master 시트에 등록된 이름과 정확히 일치해야 합니다)
  5. 로그인 버튼을 클릭합니다.
ℹ️ 로그인 후 새로고침해도 세션이 유지됩니다. 직접 로그아웃 버튼을 누르기 전까지는 재로그인이 필요 없습니다.

📄 1학기 사정안 작성

로그인 후 1학기 사정안 탭이 기본으로 열립니다.

1. 재적 현황 재적: 학급 전체 학생 수
사정인원: 실제 성적 산출 대상 학생 수
출석일수 미달자: 해당자가 없으면 "해당없음" 입력
비고: 특이사항 입력 (없으면 공란)
2. 표창 대상자 모범 / 노력 / 선행 / 봉사 / 리더십 5개 부문
드롭다운에서 학생 이름 선택 → 추천 사유 입력 → 추천교사 이름 입력
※ 학생 목록은 student_list 시트에서 자동으로 불러옵니다

📄 2학기(학년말) 사정안 작성

관리자가 2학기로 설정한 경우 상단 탭에서 2학기(학년말) 사정안 탭이 활성화됩니다.

1. 재적 현황 1학기와 동일하게 재적·사정인원·미달자·비고를 입력합니다.
2. 표창 대상자 1학기와 동일하게 5개 부문 표창 대상자를 입력합니다.
3. 개근자 명단: 쉼표로 구분하여 입력 (예: 김철수,이영희,박민준)
개근자 수: 명단 입력 후 자동 계산됩니다
※ 3학년은 "3년 개근자", 1·2학년은 "1년 개근자"로 자동 표시됩니다
⚠️ 관리자가 학기를 1학기로 설정해 두면 2학기 탭이 잠겨 접근할 수 없습니다.

💾 저장 · 제출 · 인쇄

💾 저장 작성 중인 내용을 임시 저장합니다. 언제든 다시 수정할 수 있습니다. 저장 후에도 제출/인쇄 버튼이 활성화됩니다.
📤 제출 최종 완성된 내용을 제출합니다. 저장을 먼저 해야 제출 버튼이 활성화됩니다. 제출 후에도 수정·재제출이 가능합니다.
🖨️ 인쇄 현재 학기 사정안을 A4 가로 형식으로 인쇄합니다. 저장 후 인쇄 버튼이 활성화됩니다.
💡 작성 도중 자리를 비워야 할 경우 저장을 눌러두면 새로고침 후에도 내용이 그대로 복원됩니다.
📖

교과 교사 매뉴얼

나이스 성적 우수자 파일을 업로드합니다

🔑 로그인

  1. 메인 화면에서 📖 과목교사 로그인 버튼을 클릭합니다.
    ⚠️버튼이 회색이면 관리자가 과목교사 로그인을 아직 열지 않은 상태입니다. 성적 산출 완료 후 관리자가 활성화합니다.
  2. 현재 학기가 표시됩니다. (관리자 설정값으로 자동 적용되며 변경 불가)
  3. 학년을 선택합니다. 선택 즉시 담당 과목 목록이 드롭다운에 채워집니다.
  4. 담당 과목을 드롭다운에서 선택합니다.
  5. 교사명을 입력합니다.
  6. 로그인 버튼을 클릭합니다.
ℹ️ 과목 목록은 과목편제 시트에 등록된 순서대로 표시됩니다. 담당 과목이 없으면 관리자에게 과목편제 등록을 요청하세요.

📁 나이스 파일 업로드

나이스(NEIS)에서 내려받은 교과 성적 우수자 xlsx 파일을 업로드합니다.

  1. 업로드 영역을 클릭하거나 파일 선택 버튼을 클릭합니다.
  2. 나이스에서 받은 .xlsx 파일을 선택합니다.
  3. 파일 내용 확인 창이 열립니다. 학년·과목명·학생 수를 검토합니다.
  4. 학생 목록이 정확하면 업로드 확인 버튼을 클릭합니다.
  5. 기존 데이터가 있는 경우 덮어쓰기 확인 창이 추가로 표시됩니다.
⚠️ 파일에서 추출된 학년과 과목명이 로그인 시 선택한 정보와 일치하지 않으면 업로드가 거부됩니다. 파일을 다시 확인하세요.
💡 같은 학년·과목 파일을 다시 올리면 기존 데이터를 덮어씁니다. 덮어쓰기 전에 기존 업로더와 시간이 표시되니 확인 후 진행하세요.

📊 업로드 현황 확인

로그인 후 화면 하단 업로드 현황 섹션에서 1·2·3학년 전체 과목의 업로드 상태를 확인할 수 있습니다.

✅ 과목명  업로드 완료 ○ 과목명  미업로드
ℹ️ 다른 교과교사가 업로드한 과목도 현황에 표시됩니다. 자신의 담당 과목만 업로드하면 됩니다.

자주 묻는 질문

사용 중 자주 발생하는 문제와 해결 방법

Q. 로그인 버튼이 회색으로 표시됩니다.

관리자가 해당 역할의 로그인을 비활성화한 상태입니다. 관리자에게 활성화를 요청하세요.

Q. 이름을 입력했는데 로그인이 안 됩니다. (담임교사)

Google Sheets class_master 시트에 등록된 이름과 정확히 일치해야 합니다. 띄어쓰기, 성과 이름 순서까지 동일하게 입력하세요. 관리자에게 시트 확인을 요청할 수 있습니다.

Q. 담당 과목 드롭다운에 과목이 없습니다. (교과교사)

관리자가 Google Sheets 과목편제 시트에 해당 학년·학기 과목을 등록하지 않은 상태입니다. 관리자에게 과목 등록을 요청하세요.

Q. 2학기 탭이 잠겨있습니다. (담임교사)

관리자 시스템 설정에서 현재 학기가 1학기로 설정되어 있습니다. 2학기 사정안 작성 기간이 되면 관리자가 2학기로 변경합니다.

Q. 데이터 전송 후 양식 시트 내용이 비어 있습니다.

담임교사들이 자료를 아직 제출하지 않았거나, 현재 탭(1학기/2학기)이 올바른지 확인하세요. 1학기 탭이 활성화된 상태에서 전송하면 양식_1학기 시트에, 2학기 탭에서 전송하면 양식_2학기 시트에 데이터가 채워집니다.

Q. 파일 업로드 후 학생 수가 0명으로 표시됩니다. (교과교사)

나이스에서 받은 파일 형식이 맞지 않을 수 있습니다. 파일의 열 구조(반·번호·이름)가 올바른지 확인하고, 문제가 지속되면 관리자에게 문의하세요.

Q. 제출한 내용을 수정할 수 있나요?

가능합니다. 로그인 후 내용을 수정하고 저장 → 제출을 다시 누르면 기존 제출 내용이 덮어씌워집니다. 관리자가 최종 데이터를 내보내기 전에는 언제든지 수정할 수 있습니다.
