물품관리현황

사용 매뉴얼

1. 개요

학교 각 관리실(교실 · 특별실 · 사무실 등)의 비품 · 물품을 등록하고 현황을 관리하는 웹 시스템입니다. 담당자마다 관리 권한이 있는 관리실이 배정되어 있어 본인이 맡은 곳의 물품만 조회 · 수정할 수 있고, 관리자는 전체 물품 현황을 확인하고 사용자 권한 관리, 데이터 다운로드까지 처리할 수 있습니다.

2. 주요 기능

  • 로그인 및 권한 구분 : 관리자(전체 조회 · 수정 · 사용자 관리), 조회관리자(전체 조회 가능하나 수정 제한), 일반 담당자(배정된 관리실만 조회 · 수정)
  • 물품 등록 : 관리실 위치 · 세부 관리실 · 담당자 · 물품분류 · 기종 · 모델명(규격) · 예산구분 · 취득일 · 수량 · 상태 · 폐기일 · 비고를 입력해 등록, 목록에 없는 품명을 직접 입력하면 다음부터 선택할 수 있도록 자동 저장됨
  • 물품 현황 조회 : 담당 관리실별 탭으로 구분되어 표시, 검색 · 필터링, 페이지 단위로 목록 확인
  • 물품 수정 · 삭제 : 여러 건을 체크박스로 선택해 한 번에 수정 저장하거나 삭제 가능, 모든 수정 · 삭제 내역은 "로그" 시트에 자동 기록됨
  • 데이터 다운로드 : 관리자 · 조회관리자는 전체 데이터를, 일반 담당자는 본인이 보고 있는 관리실 데이터를 CSV 파일로 다운로드
  • 관리자 화면 : 사용자(담당자) 목록 조회 · 추가 · 권한(담당 관리실) 수정
  • 내 정보 수정 : 본인 비밀번호 변경

3. 사용 방법

  1. 로그인 화면에서 아이디와 비밀번호를 입력합니다.
  2. 로그인하면 본인이 담당하는 관리실 탭(관리자는 "전체")이 표시되며, 탭을 눌러 해당 관리실의 물품 목록을 확인합니다.
  3. 새 물품을 등록하려면 등록 화면에서 관리실 위치 · 세부 관리실 · 물품분류 · 품명 등을 입력하고 등록합니다.
  4. 기존 물품을 수정하려면 목록에서 해당 행의 체크박스를 선택해 수정 모드로 전환한 뒤 내용을 고치고 저장합니다.
  5. 삭제할 물품은 체크박스로 선택한 뒤 삭제 버튼을 누릅니다.
  6. 현재 보고 있는 관리실(또는 전체) 데이터를 CSV로 받으려면 다운로드 버튼을 누릅니다.
  7. 관리자는 관리자 화면에서 사용자 목록을 확인하고 담당 관리실 권한을 추가 · 수정할 수 있습니다.
  8. 본인 비밀번호를 바꾸려면 "내 정보" 화면에서 현재 비밀번호 확인 후 새 비밀번호를 저장합니다.

4. 코드 구조

Google Apps Script 프로젝트로 구성되며, code.gs(서버 로직)와 로그인 · 대시보드 · 등록 · 관리자 화면을 모두 담은 index.html 한 파일로 이루어져 있습니다. "물품현황" 시트가 실제 물품 데이터를, "사용자" 시트가 로그인 계정과 담당 관리실 권한을, "관리실" 시트가 위치별 세부 관리실 목록을, "물품코드" 시트가 분류별 품명 목록을 관리합니다. 물품이 수정 · 삭제될 때마다 "로그" 시트에 이력이 자동으로 남습니다.

5. 주의사항

  • 담당자 권한(담당 관리실)은 이름과 비밀번호가 모두 일치해야 추가되며, 같은 이름이라도 비밀번호가 다르면 권한 추가가 거부됩니다.
  • 물품을 여러 건 삭제할 때는 행 번호가 큰 순서부터 처리되어 삭제 도중 목록이 꼬이지 않지만, 삭제 후에는 되돌릴 수 없으므로 신중하게 선택해야 합니다.
  • 다운로드되는 CSV는 현재 화면에 표시된 관리실(또는 전체) 기준이므로, 원하는 범위의 탭을 선택한 후 다운로드해야 합니다.
  • 조회관리자는 전체 데이터를 볼 수 있지만 수정 권한은 제한되어 있습니다.
