📚 공부 인증 앱

사용 매뉴얼

1. 개요

학급 학생들이 각자의 공부 스케줄을 등록하고, 정해진 시간에 공부하는 모습을 사진으로 인증해 학급 전체와 실시간으로 공유하는 모바일 웹앱입니다. 서로의 인증 피드를 확인하고 댓글로 응원하며 꾸준한 학습 습관을 만들 수 있도록 돕습니다. 로그인 절차 없이 이름과 컬러만 정하면 바로 사용할 수 있습니다.

2. 주요 기능

  • 간편 시작 : 이름과 개인 컬러를 선택하면 로그인 없이 바로 사용 (다음 접속부터는 자동으로 로그인)
  • 요일별 공부 스케줄 등록 : 공부하는 요일을 선택하고, 요일마다 시작~종료 시간을 지정
  • 공부 인증 : 공부하는 모습을 사진으로 찍거나 선택하고 한마디 메모를 남겨 인증 게시 (사진은 자동으로 리사이즈되어 업로드됨)
  • 실시간 피드 : 반 전체의 인증 게시물을 실시간으로 확인 (15초 간격 자동 새로고침 백업 포함)
  • 댓글 : 각 인증 게시물에 댓글을 달아 서로 응원
  • 학급 스케줄 한눈에 보기 : 친구들의 요일별 공부 시간과 "오늘 공부하는 친구" 표시 확인
  • 계정 삭제 : 본인의 스케줄 · 인증샷 · 댓글을 한 번에 삭제

3. 사용 방법

  1. 처음 접속하면 이름을 입력하고 나만의 컬러를 선택합니다.
  2. 공부하는 요일을 선택하고 요일별 시작~종료 시간을 설정한 뒤 "시작하기"를 누릅니다. (스케줄은 나중에 설정해도 됨)
  3. 하단 탭에서 피드를 눌러 반 친구들의 인증 현황을 실시간으로 확인합니다.
  4. 인증하기 탭에서 공부하는 모습을 사진으로 찍거나 선택하고, 한마디를 남긴 뒤 "인증 완료"를 누릅니다.
  5. 스케줄 탭에서 내 스케줄을 언제든 수정하거나 친구들의 스케줄을 확인할 수 있습니다.
  6. 더 이상 사용하지 않을 경우 스케줄 탭 하단의 "내 계정 삭제"로 내 데이터를 정리할 수 있습니다.

4. 코드 구조

단일 HTML 파일로 구성되어 있으며, Firebase Realtime Database(gonbuhaja 프로젝트)를 데이터 저장소로 사용합니다. 화면은 이름 입력 → 스케줄 설정 → 메인 화면 순서로 전환되고, 메인 화면은 피드 / 인증하기 / 스케줄 3개의 탭으로 구성됩니다. 모든 로직은 하나의 <script> 블록 안에 JavaScript로 작성되어 있습니다.

5. 주의사항

  • 별도 로그인이 없어 이름만으로 사용자를 구분하므로, 같은 이름을 여러 명이 사용하면 데이터가 섞일 수 있습니다.
  • 인증 피드는 최근 100개까지만 보관되며, 그 이전 게시물은 자동으로 삭제됩니다.
  • 사진 데이터가 Firebase Realtime Database에 직접 저장되므로, 이미지가 너무 크면 업로드에 실패할 수 있습니다.
  • 계정 삭제 시 스케줄 · 인증샷 · 댓글이 모두 삭제되며 복구할 수 없습니다.
