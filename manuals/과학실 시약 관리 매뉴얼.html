과학실 시약 관리

사용 매뉴얼

1. 개요

과학실에서 사용하는 산 · 염기 · 유기 · 무기 시약의 재고를 구글 시트와 실시간으로 연동해 관리하는 웹 프로그램입니다. 사용자는 시약을 사용할 때마다 사용량을 기록하면 잔량이 자동으로 차감되고, 관리자는 PIN 번호로 별도 화면에 접속해 시약 등록, 사용 기록 · 재고 조회, 수정 · 삭제까지 처리할 수 있습니다.

2. 주요 기능

일반 사용자

  • 시약 사용 기록 : 날짜 · 분류(산/염기/유기/무기) · 시약 · 사용량 · 사용자 이름을 입력해 제출하면 "사용기록" 시트에 기록되고 해당 시약의 잔량이 자동으로 차감됨
  • 분류를 선택하면 해당 분류의 시약 목록과 현재 잔량이 실시간으로 드롭다운에 표시됨

관리자 (PIN 번호 5자리로 접속)

  • 새 시약 추가 : 분류 · 순번 · 시약명 · 화합물 · 용량 · 초기량을 입력하면 구글 시트에 즉시 추가됨(초기 잔량 = 초기량)
  • "사용기록 / 산 / 염기 / 유기 / 무기" 탭별로 구글 시트 데이터를 표로 확인
  • 각 행 수정(잔량 또는 사용량 값 변경) 및 삭제(구글 시트에서도 함께 삭제)
  • "구글 스프레드시트 바로가기" 버튼으로 연결된 원본 시트를 새 탭에서 바로 열람

3. 사용 방법

일반 사용자

  1. 사용 날짜를 확인합니다. (기본값: 오늘 날짜)
  2. 시약 분류를 선택합니다.
  3. 사용한 시약을 드롭다운에서 선택합니다. (현재 잔량이 함께 표시됨)
  4. 사용량(g/mL)과 사용자 정보(예: 학번 이름)를 입력합니다.
  5. "기록 제출하기"를 누르면 사용 기록이 저장되고 잔량이 자동으로 차감됩니다.

관리자

  1. "⚙️ 관리자 모드 접속" 버튼을 누르고 PIN 번호(5자리)를 입력합니다.
  2. 새 시약을 등록하려면 "새 시약 추가" 영역에 분류 · 순번 · 시약명 등을 입력하고 "구글 시트에 즉시 추가"를 누릅니다.
  3. 상단 탭(사용기록/산/염기/유기/무기)을 눌러 해당 데이터를 표로 확인합니다.
  4. 특정 행의 값을 고치려면 "수정" 버튼을 눌러 새 수치를 입력하고, 행을 지우려면 "삭제" 버튼을 누릅니다. (구글 시트에도 즉시 반영됨)
  5. "📊 구글 스프레드시트 바로가기"로 원본 시트를 직접 열어 확인할 수 있습니다.
  6. 작업이 끝나면 "🚪 로그아웃" 버튼으로 사용자 화면으로 돌아갑니다.

4. 코드 구조

Google Apps Script 프로젝트로 구성되며, code.gs(서버 로직)와 index.html(사용자 화면과 관리자 화면을 모두 담은 단일 페이지) 두 파일로 이루어져 있습니다. 시약 분류(산/염기/유기/무기)별로 별도의 구글 시트를 사용하며, 사용 기록은 사용기록 시트에 따로 쌓입니다. getReagentList(시약 목록 조회), submitUsage(사용 기록 및 잔량 차감), addReagentByAdmin(시약 추가), getSheetDataForAdmin(관리자용 데이터 조회), updateReagentByAdmin(수정 · 삭제) 함수가 각각의 기능을 담당합니다.

5. 주의사항

  • 관리자 PIN 번호는 코드에 고정되어 있으므로, 변경하려면 코드를 직접 수정해야 합니다.
  • 관리자 모드에서 행을 삭제하면 구글 시트에서도 즉시 삭제되므로 신중하게 확인 후 진행해야 합니다.
  • 시약 분류(산/염기/유기/무기)마다 별도 시트를 사용하므로, 새 시약을 추가할 때 분류를 정확히 선택해야 올바른 시트에 저장됩니다.
  • 잔량은 시트에 직접 저장되는 값이므로, 여러 사람이 동시에 같은 시약을 기록하면 차감 순서에 따라 잔량 계산이 어긋날 수 있습니다.
