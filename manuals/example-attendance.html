출석부 자동 정리 프로그램

사용 매뉴얼 (실제 프로그램 등록 시 이 예시는 교체됩니다)

제작 교사수학과 · 김OO 교사
사용 언어/도구Python, openpyxl
실행 환경Windows 10 이상, Python 3.10
원본 위치programs/example-attendance

1. 개요

매일 손으로 정리하던 학급 출석부를 엑셀 파일로 업로드하면, 학생별 결석·지각 횟수를 자동으로 집계하고 학급 리포트를 생성해주는 프로그램입니다.

2. 주요 기능

  • 엑셀 출석부 업로드 후 결석/지각/조퇴 자동 집계
  • 학생별·학급별 통계 리포트(xlsx) 자동 생성

3. 사용 방법

  1. programs 폴더의 실행 파일을 더블클릭해 실행합니다.
  2. 정리할 출석부 엑셀 파일을 선택합니다.
  3. 완료되면 같은 폴더에 결과 리포트 파일이 생성됩니다.

4. 코드 구조

이 항목은 실제 코드를 분석해 주요 파일과 역할을 정리하는 자리입니다. (예: main.py 실행 진입점, report.py 리포트 생성 로직)

5. 주의사항

이 페이지는 대시보드 디자인을 보여주기 위한 예시입니다. 실제 프로그램 폴더가 등록되면 코드를 분석해 내용을 새로 작성합니다.
