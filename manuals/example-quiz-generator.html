AI 시험문제 생성기

사용 매뉴얼 (외부 URL 링크 방식 예시)

제작 교사영어과 · 박OO 교사
바로가기 방식URL (외부에 이미 호스팅된 웹앱으로 바로 연결)
바로가기 주소https://example.com

1. 개요

웹에서 바로 사용할 수 있는 AI 기반 시험 문제 생성 도구입니다. 별도 설치 없이 브라우저에서 바로 접속해서 사용합니다. 이 페이지는 "바로가기유형"을 폴더가 아니라 URL로 설정했을 때 카드가 외부 링크로 바로 연결되는 것을 보여주는 예시입니다.

2. 주요 기능

  • 과목/단원을 입력하면 객관식·주관식 문제를 자동 생성
  • 생성된 문제를 엑셀/워드 형식으로 내보내기

3. 사용 방법

  1. 대시보드 카드의 "바로가기" 버튼을 클릭해 사이트로 이동
  2. 과목과 문제 유형을 선택 후 생성 버튼 클릭
  3. 생성된 문제를 검토 후 다운로드

4. 주의사항

이 페이지는 URL 방식 카드가 어떻게 보이는지 보여주기 위한 예시입니다. 실제 외부 도구를 등록할 때는 바로가기값에 실제 서비스 주소를 입력하면 됩니다.
