프로그램 추가

새 프로그램을 입력해 목록에 추가하고, 다 끝나면 한 번에 programs.csv로 내려받습니다.

1. 현재 데이터 불러오기

불러오는 중입니다...

같은 폴더의 programs.csv를 자동으로 불러오지 못했습니다. 아래에서 직접 선택해주세요.

2. 프로그램 정보

지금 기존 항목을 수정하고 있습니다. 아래 "목록에 반영" 버튼을 누르면 그 항목이 수정됩니다.
카드 상단에 표시할 짧은 태그. 비워두면 표시 안 됨
카드에 그대로 보여지는 2~3문장 설명
programs 폴더 안 파일명만 입력 (예: 내프로그램.html)
manuals 폴더 안에 있는 매뉴얼 html 파일명만 입력

3. 전체 목록

각 행의 "수정"을 누르면 위 폼에 내용이 채워집니다. "삭제"는 바로 목록에서 빠집니다. 아직 저장 전이며, 최종 다운로드 전까지는 언제든 되돌릴 수 있어요.
